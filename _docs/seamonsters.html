<subtitle>
At the first Icon Salon last night our theme was Sea Monsters. Imagine that San Francisco is now underwater and we need new symbols to indicate the dangers of the new monsters that might be lurking in the city.Icon Salons are occasional get-togethers for Icon Makers. These sketches were created by these people for the public domain using a process derived from the Iconathon and open-sourced as Icon Camp. The intention is that any symbols created from these sketches should be given back the public domain.Next steps: Make more sea monster icons. Cut them out.
